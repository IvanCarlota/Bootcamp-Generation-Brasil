Contêineres

Os Contêineres são os elementos de leiaute mais básicos no Bootstrap e são usados para conter, preencher e (às vezes) centralizar o conteúdo dentro deles.

Por padrão o Bootstrap estabelece os pontos de quebra (media query) para alguns tamanhos de tela/janela.

≥576px 		Medium
