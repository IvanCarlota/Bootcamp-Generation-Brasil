A reinicialização baseia-se apenas na padronização de estilo para alguns elementos do HTML, fazendo o uso de seletores CSS pelo nome de tags e atributos.

Parágrafos

O elemento HTML <p> (ou elemento de parágrafo) representa um parágrafo do texto.


Listas

O elemento HTML <ul> (ou elemento de lista desordenada) representa uma lista de itens sem ordem de dígito, isto é, uma coleção de itens que não trazem uma ordenação numérica e as suas posições.

  • Banana
    • Nanica
    • Prata
  • Abacaxi
  • Graviola

O elemento HTML <ol> (ou elemento de lista ordenada) representa uma lista de itens ordenados de forma numérica.

  1. Banana
  2. Abacaxi
  3. Graviola

o elemento HTML <dl> (ou o elemento de lista de definições) engloba uma lista de pares de termos de descrições.

Textos pré-formatados

O elemento HTML <pre> representa o texto pré-formatado que deve ser apresentado exatamente como esta escrito no arquivo HTML. 

        ***********************
        * TEXTO PRÉ-FORMATADO *
        ***********************
